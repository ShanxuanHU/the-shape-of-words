# 单词的形状

英语单词并不是一座座孤岛。

它们有时共享同一个词根，有时从相近的概念出发，沿着不同语言的路径生长，最终形成一张彼此呼应的词汇网络。

《单词的形状》是一个词汇学习系列。每篇文章选择一组相关联的单词，以词根词缀为主线，尝试回答：

> 一个单词是怎样长成今天这个样子的？

## 内容方向

- **主线**：从词根和词缀出发，梳理英语单词家族。
- **支线**：探索读音、拼写、语义变化和文化背景带来的关联。
- **番外**：偶尔走进其他语言，看看相似的概念如何用不同方式表达。

文章会同步发布在 GitHub 和微信公众号。GitHub 用于沉淀内容、修订错误和持续补充；微信公众号用于日常阅读和分享。

## 阅读方式

你不需要一次记住整张词表。更重要的是抓住词汇之间的方向感：

- `inspect`：向里仔细看，于是有了“检查”。
- `prospect`：向前看，于是有了“前景”。
- `retrospect`：向后看，于是有了“回顾”。
- `circumspect`：向四周看，于是有了“谨慎”。

当一个陌生单词再次出现时，你或许会发现，它并不完全陌生。

## 读音标注

文章中的正文词条会附带一套简化的英文拆音标注。这种方法通常叫作 **pronunciation respelling**：使用容易联想到发音的英文字母组合来提示读音，不要求读者事先掌握国际音标。

- 连字符用于拆分音节，例如 `in-SPEKT`。
- 大写部分表示重读音节，例如 `spek-TAK-yuh-ler`。
- 少数单词会因为词性不同而改变重音。例如，`suspect` 作动词时读作 `suh-SPEKT`，作名词时读作 `SUH-spekt`。

拆音标注比中文谐音更准确，也比音标更容易上手，但仍然只是一种近似提示。学习读音时，最好配合词典音频跟读。

## 一词多义

不少常用单词会沿着不同方向延伸出新的含义。为了保持主词条简洁，文章不会把所有义项都堆在单词后的括号里。

遇到值得掌握的常见义项时，文章会增加一条英文例句，并在例句后单独说明该词在当前语境中的意思。例如：

> The salesperson called a promising **prospect** after lunch.
>
> （这里 **prospect** 指“潜在客户”。）

这种编排方式希望让读者先在语境中遇见含义，再理解词义之间的联系。
